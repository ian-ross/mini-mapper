# PWM shell application

Sets up 5 kHz PWM timers on:

 * TIM10: PB8
 * TIM3: PA6, PA7, PC8, PC9

## Commands

pwm1 {on|off}
pwm4 {on|off}

## Variables

pwm1-state
pwm1-polarity
pwm1-duty
pwm1-reload

pwm4-state
pwm4-polarity-1
pwm4-polarity-2
pwm4-polarity-3
pwm4-polarity-4
pwm4-duty-1
pwm4-duty-2
pwm4-duty-3
pwm4-duty-4
pwm4-reload
